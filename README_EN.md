Cybersecurity Learning Portfolio



Welcome to my cybersecurity learning portfolio.

This repository shows my hands-on progress in Linux, networking, system security, and defensive/offensive security.



The goal of this repository is to build strong practical skills in cybersecurity through labs, scripts, and real-world projects.



Learning Path



My learning journey follows these main steps:



Technical Foundations

Understand how systems and networks work (Linux, networking basics, virtualization).



System \& Network Security

Learn how to secure systems and networks, and identify common vulnerabilities.



Offensive Security

Understand how attacks work in order to better defend systems.



Defense \& SOC

Analyze logs, detect suspicious activity, and understand defensive tools.



Automation \& Scripting

Write scripts to automate security tasks and analysis.



Practice \& Projects

Complete practical projects and document real security workflows.
